# TypeScript

This project contains TypeScript exercises for learning and practicing TypeScript fundamentals.

## Requirements

- Node.js (v12+)
- npm

## Project Structure

```
TypeScript/
  task_0/
    js/
      main.ts    # Student interface, studentsList, and table rendering
    package.json
    tsconfig.json
    webpack.config.js
```

## Task 0 - Student Table

Defines a `Student` interface and renders a table of students using Vanilla JS DOM manipulation.

### Setup

```bash
cd TypeScript/task_0
npm install
```

### Build

```bash
npm run build
```

### Run development server

```bash
npm run start-dev
```

Open the browser and you will see a table with 2 rows displaying student first names and locations.
